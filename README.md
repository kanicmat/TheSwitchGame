If the user makes it to the end of the count down, all LEDs turn on, their score will remain visible and the timer will stay at 0 until the reset button is pressed. When the reset button is pressed the whole board resets, meaning that the users score, and the timer will reset to their initial values of 0 and 20 respectively. Additionally, if the game ends because of the user flipping the incorrect switch, the game can be reset using the reset button, which in this case will override the stop timer case and reset the high score and timer to their original values.
